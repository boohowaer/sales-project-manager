# 批量导入模板文件

本目录包含销售项目管理系统的批量导入模板文件。

## 📁 文件列表

### 模板文件
- `customers_template.csv` - 客户信息导入模板
- `projects_template.csv` - 项目信息导入模板

### 说明文档
- `../导入模板说明.md` - 详细的导入说明文档

## 🚀 快速开始

### 1. 导入客户
1. 下载 `customers_template.csv` 模板
2. 按照模板格式填写客户信息
3. 在系统中上传填写好的CSV文件

### 2. 导入项目
1. **重要**: 确保已先导入对应的客户数据
2. 下载 `projects_template.csv` 模板
3. 按照模板格式填写项目信息
4. 在系统中上传填写好的CSV文件

## ⚠️ 重要提示

1. **导入顺序**: 必须先导入客户，再导入项目
2. **文件编码**: 使用UTF-8编码，避免中文乱码
3. **日期格式**: 统一使用 YYYY-MM-DD 格式
4. **客户关联**: 项目中的"客户名称"必须与系统中的客户名称完全一致

## 📖 详细说明

请查看 [导入模板说明.md](../导入模板说明.md) 获取完整的字段说明、验证规则和示例。

## 💡 使用技巧

1. **Excel用户**: 
   - 打开CSV文件时选择UTF-8编码
   - 保存时选择 "CSV UTF-8 (逗号分隔)(*.csv)" 格式

2. **数据验证**:
   - 导入前先检查必填字段是否填写完整
   - 日期格式是否正确（YYYY-MM-DD）
   - 项目中的客户名称必须与系统中的客户名称完全一致
   - 客户来源填写数据字典中配置的标签名称

3. **批量导入建议**:
   - 每次导入不超过1000条记录
   - 先用少量数据测试导入流程
   - 保留原始数据备份
