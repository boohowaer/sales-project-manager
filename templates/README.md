# 批量导入模板文件

本目录包含销售项目管理系统的批量导入模板文件。

## 📁 文件列表

### 模板文件
- `customers_template.csv` - 客户信息导入模板（CSV格式）
- `customers_template.xlsx` - 客户信息导入模板（Excel格式）
- `projects_template.csv` - 项目信息导入模板（CSV格式）
- `projects_template.xlsx` - 项目信息导入模板（Excel格式）

### 说明文档
- `../导入模板说明.md` - 详细的导入说明文档

## 🚀 快速开始

### 导入顺序
**重要**: 必须先导入客户，再导入项目！因为项目需要关联客户。

### 1. 导入客户
1. 下载客户导入模板（CSV或Excel格式）
2. 按照模板格式填写客户信息
3. 在系统中上传填写好的文件
4. 点击"开始导入"

### 2. 导入项目
1. **重要**: 确保已先导入对应的客户数据
2. 下载项目导入模板（CSV或Excel格式）
3. 按照模板格式填写项目信息
4. 在系统中上传填写好的文件
5. 点击"开始导入"

## ⚠️ 重要提示

### 文件格式支持
- ✅ **CSV格式** (.csv) - 通用格式，任何文本编辑器都可以打开
- ✅ **Excel格式** (.xlsx, .xls) - 推荐使用，Excel可以直接编辑

### 必填字段标记
模板中带 `*` 符号的字段为必填字段：
- 客户导入：**客户名称***
- 项目导入：**项目名称***、**客户名称***

### 日期格式支持
系统支持多种日期格式，会自动转换：
- ✅ `2026-01-15`（标准格式）
- ✅ `2026/1/15`（斜杠格式）
- ✅ `2026年1月15日`（中文格式）
- ✅ Excel中的任何有效日期格式

**提示**: 使用Excel编辑时，可以直接输入日期，Excel会自动格式化。

### 数据去重规则
导入时会自动去重，只导入新数据：

**客户去重**: 客户名称 + 公司名称 同时重复才算重复
**项目去重**: 项目名称 + 客户名称 同时重复才算重复

**重复数据处理**: 在重复组内选择字段信息最完整的那一条导入，其余跳过。

### 字段填写建议

#### 客户信息
- **客户名称***: 使用人名（如：张三、李四）
- **公司名称**: 客户所在公司的全称
- **邮箱**: 有效的邮箱地址
- **电话**: 支持任意格式的电话号码
- **备注**: 客户的其他补充信息

#### 项目信息
- **项目名称***: 简洁明确的项目名称
- **客户名称***: 必须是系统中已存在的客户名称
- **项目描述**: 项目的详细信息
- **项目状态**: active(进行中) / won(已成交) / lost(已丢失) / on_hold(暂停)
- **项目金额**: 只填数字，不含货币符号和千分位（如：50000）
- **成功概率**: 0-100之间的整数（如：75）
- **归属年份**: 项目归属的年份（如：2026）

## 🔍 常见错误及解决方案

| 错误信息 | 原因 | 解决方案 |
|---------|------|---------|
| "客户名称不能为空" | 项目记录中客户名称字段为空 | 检查CSV文件，填写客户名称 |
| "客户不存在" | 项目关联的客户名称在系统中找不到 | 先导入客户，或检查客户名称拼写是否一致 |
| "日期格式不正确" | 日期格式无法识别 | 使用标准格式（YYYY-MM-DD）或用Excel编辑 |
| "项目状态无效" | 状态字段值不在允许的范围内 | 使用 active/won/lost/on_hold 之一 |
| "缺少必填列" | CSV文件的表头缺少必填字段 | 确保表头包含所有带 * 标记的字段 |

## 💡 使用技巧

### Excel用户
1. 打开模板文件
2. 直接在Excel中编辑数据
3. Excel会自动处理日期格式
4. 保存时选择 "CSV UTF-8" 或直接保存为xlsx格式

### 批量导入建议
1. **每次导入不超过1000条记录**
2. **先用少量数据测试导入流程**
3. **保留原始数据备份**
4. **导入前检查必填字段是否填写完整**
5. **确保客户名称与系统中的完全一致**

## 📊 导入结果反馈

导入完成后会显示详细信息：
- **成功导入**: 实际导入到系统的新数据条数
- **跳过**: 已存在或重复的数据条数
- **总计**: CSV/Excel文件中的总条数

### 反馈示例
```
✅ 成功导入 5 条新客户数据，跳过 2 条已存在的客户（总计 7 条记录）
✅ 成功导入 3 条新项目数据（总计 3 条记录）
⚠️ 所有客户都已存在，跳过导入
```
